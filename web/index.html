The result

Tip reduction is real, measured, and honestly reported

The optimizer and the baseline are produced by the same compiler from the same intermediate representation — the baseline is simply every optimization pass turned off (one fresh tip per transfer). The comparison is therefore apples-to-apples and cannot be cherry-picked. Tips and steps are exact counts; wall-clock is a clearly-labelled model estimate.

Why you can trust the number

Correctness is a guarantee, not a hope

Cutting tips is worthless if it changes what liquid ends up where. PipetteC makes two properties machine-checkable and tests them over hundreds of randomly generated specs:

Delivery-equivalence

  • The optimized protocol delivers the same total volume for every (source → destination) pair as the naive one.
  • Verified over 550 randomized specs — 0 counterexamples.

Contamination-safety

  • A tip is reused across two transfers only if they share a declared tip class.
  • An unsafe reuse is a validator error, not a silent optimization.

How it works

Built like a compiler, not a script

spec.yaml / Echo .csv→ TransferGraph IR→ optimization passes→ static validator→ OT-2 codegen→ opentrons_simulate ✓
1

One IR, two front-ends

A human-written YAML template and a real Echo picklist CSV both lower to the same intermediate representation.

2

Optimization as passes

Multi-channel packing, source-grouped reordering, tip reuse, and reagent batching — each independently benchmarked.

3

Simulator as the gate

Every generated protocol must exit 0 on Opentrons' own opentrons_simulate, enforced in CI on every push.

Honest positioning

The contribution is execution, not a novel algorithm

Compilers for liquid handling exist (Roboliq), and tip minimization is published operations research (capacitated vehicle-routing / linear-programming formulations). PipetteC applies and cites that prior work — it does not claim to invent it. What is distinctive is the assembled artifact: an OT-2 target, an Echo-picklist front-end, an optimizing IR, and the official simulator wired in as a continuous-integration gate — legible and runnable in minutes.

Prior art cited in the repository: Roboliq / PR-PR (ACS Synth. Biol. 2018); tip-minimization as CVRP/LP (NAR Genomics & Bioinformatics 2022; Digital Discovery 2025); PyLabRobot (2023).
